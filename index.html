Vehicular Rental System

Welcome to the Vehicular Rental System website.

Features

  • User Registration
  • User Login
  • View Available Vehicles
  • Vehicle Booking
  • Bill Generation

Backend is implemented using Java and MySQL.

Project Execution & Integration

This project is implemented using a hybrid approach. The frontend of the Vehicular Rental System is hosted online using GitHub Pages, which provides a public website link.

The backend logic such as user registration, vehicle viewing, booking, and bill calculation is implemented using Java and MySQL through JDBC in Eclipse.

Backend Execution

Backend operations are executed locally in Eclipse. Booking details are stored in the MySQL database and displayed in the console output.
